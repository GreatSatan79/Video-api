# Video-api
<p>这是一个可解析视频的API</p>

## 功能如下

<ul>
  <li>
    <p>电脑访问时会激活DPlayer播放器<p>
  </li>
  <li>
    <p>手机访问时会激活Video.js播放器<p>
  </li>
</ul>

## 如何使用?
<p>上传至你网站的根目录,然后在浏览器地址栏输入&nbsp;&nbsp;<code>http://你的网站/video/?url=视频地址</code>确认能否播放</p>
<p>然后在你需要用到的地方插入&nbsp;&nbsp;<code><iframe id="iframe" class="iframe" width="100%" height="100%" frameborder="0" scrolling="no" allowfullscreen="true" src="http://你的网站/video/?url=视频地址"></iframe></code></p>
